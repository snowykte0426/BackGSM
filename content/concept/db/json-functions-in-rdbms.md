## 왜 JSON 함수가 필요한가?

관계형 DB에서도 반정형 데이터를 부분적으로 다뤄야 하는 경우가 있기 때문입니다.

## 대표 활용

- JSON 필드 조회
- 특정 경로 추출
- 조건 검색
- 부분 업데이트

## 제품별 차이

- MySQL/MariaDB/PostgreSQL/Oracle/MSSQL 모두 JSON 지원 방향이 다름
- 함수 이름과 인덱싱 전략도 차이가 있음

## 면접 포인트

- JSON을 지원한다고 해서 NoSQL이 되는 것은 아니며, 여전히 관계형 DB의 맥락 안에서 이해해야 합니다.
