## CommonJS

- `require`, `module.exports`
- 전통적인 Node.js 모듈 시스템

## ESM

- `import`, `export`
- JavaScript 표준 모듈 시스템

## 왜 중요한가?

- 빌드 설정
- 런타임 호환성
- tsconfig/module 설정

## 면접 포인트

- TypeScript 백엔드에서는 모듈 시스템 차이가 실행/빌드 문제로 자주 이어지므로 실무적으로 중요합니다.
