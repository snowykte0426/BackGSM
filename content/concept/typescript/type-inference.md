## 타입 추론이란?

개발자가 모든 타입을 직접 쓰지 않아도 TypeScript가 문맥을 보고 타입을 유추하는 기능입니다.

예:

```ts
const count = 3
// number로 추론
```

## 추론이 잘 되는 경우

- 변수 초기값
- 함수 반환값
- 배열과 객체 리터럴
- 제네릭 함수 호출 시 인자 기반 추론

## 추론만 믿어도 될까?

항상 그렇진 않습니다.

- 공개 API
- 복잡한 제네릭
- 의도를 명확히 드러내야 하는 타입

이런 경우는 명시적 타입 선언이 더 좋을 수 있습니다.

## 면접 포인트

- TypeScript는 타입을 "안 써도 된다"가 아니라, 필요한 곳은 명시하고 나머지는 추론을 활용하는 균형이 중요합니다.
