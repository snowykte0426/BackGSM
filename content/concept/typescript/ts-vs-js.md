## JavaScript

- 동적 타입 언어
- 런타임에 타입 오류가 드러나는 경우가 많음

## TypeScript

- JavaScript 위에 정적 타입 시스템을 얹은 언어
- 컴파일 시점에 더 많은 오류를 잡을 수 있음

## 장점

- 리팩터링 안정성
- IDE 지원 강화
- 대규모 코드베이스 유지보수에 유리

## 면접 포인트

- TypeScript는 JavaScript를 대체한다기보다, 더 안전하게 개발하기 위한 상위 집합이라고 설명하면 좋습니다.
