## Node.js와 TypeScript

TypeScript 백엔드는 보통 Node.js 런타임 위에서 동작합니다.

## 왜 TypeScript를 백엔드에 쓰나?

- 타입 안정성
- 리팩터링 용이성
- 대규모 코드베이스 유지보수성 향상

## 자주 쓰는 영역

- API 서버
- BFF
- 백그라운드 워커
- 서버리스 함수

## 면접 포인트

- TypeScript 백엔드는 프론트와 언어를 통일한다는 장점도 있지만, 타입 안정성과 생산성 측면도 크다고 설명하면 좋습니다.
