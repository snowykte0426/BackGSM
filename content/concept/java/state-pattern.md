## 상태 패턴이란?

객체의 내부 상태에 따라 동작이 달라질 때,
상태를 별도 객체로 분리하는 패턴입니다.

## 언제 쓰나?

- 상태별 분기가 복잡할 때
- `if-else`나 `switch`가 상태마다 커질 때

## 예시

- 주문 상태
- 문서 승인 상태
- 게임 캐릭터 상태

## 면접 포인트

- 전략 패턴과 비슷해 보이지만, 상태 패턴은 객체의 내부 상태 변화와 함께 동작이 달라진다는 점이 다릅니다.
