## Servlet이란?

Servlet은 Java 기반 웹 애플리케이션에서 HTTP 요청을 받아 처리하고 응답을 만드는 서버 측 기술입니다.

## 역할

- 요청 수신
- 요청 데이터 파싱
- 비즈니스 로직 호출
- 응답 생성

## 왜 중요한가?

Spring MVC 같은 프레임워크도 결국 Servlet 기반 위에서 동작합니다.

## 면접 포인트

- Servlet은 Java 웹 기술의 기본 토대라는 점을 설명하면 좋습니다.
