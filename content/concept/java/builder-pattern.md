## 빌더 패턴이란?

복잡한 객체 생성 과정을 단계적으로 구성할 수 있게 하는 패턴입니다.

## 왜 쓰나?

- 생성자 파라미터가 많을 때
- 가독성을 높이고 싶을 때
- 불변 객체 생성에 유리할 때

## 장점

- 어떤 값이 어떤 필드인지 읽기 쉬움
- 선택적 파라미터 처리에 유리

## 면접 포인트

- 점층적 생성자 문제를 해결하는 대표 패턴이라고 설명하면 좋습니다.
