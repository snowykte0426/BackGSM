## Kotest

Kotlin 친화적인 테스트 프레임워크입니다.

특징:
- 풍부한 테스트 스타일
- matcher 지원
- 데이터 기반 테스트 작성 편의성

## MockK

Kotlin 친화적인 mocking 라이브러리입니다.

특징:
- Kotlin 문법과 잘 맞음
- final class, object mocking 지원이 편함

## 언제 좋은가?

- Kotlin 프로젝트
- 가독성 높은 테스트 DSL이 중요할 때

## 면접 포인트

- JUnit이 기본 프레임워크라면, Kotest는 Kotlin 친화성 측면에서 강점이 있다고 설명할 수 있습니다.
