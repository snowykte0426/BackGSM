## Spring

Spring은 Java 애플리케이션 개발을 위한 프레임워크입니다.
IoC, DI, AOP, 트랜잭션 처리 같은 핵심 기능을 제공합니다.

## Spring Boot

Spring Boot는 Spring을 더 빠르고 쉽게 시작할 수 있게 해주는 도구입니다.

주요 특징:
- 자동 설정(Auto Configuration)
- 내장 서버 제공
- starter 의존성 제공
- 운영/배포 편의성 향상

## 핵심 차이

- Spring: 핵심 프레임워크
- Spring Boot: Spring 기반 개발 생산성을 높여주는 확장 도구

## 면접 포인트

- Spring Boot가 Spring을 대체하는 것이 아니라, Spring을 더 쉽게 쓰도록 돕는다고 설명하면 좋습니다.
