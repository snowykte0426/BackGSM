## WAS란?

WAS는 동적인 웹 애플리케이션 요청을 처리하는 서버입니다.
정적 파일만 전달하는 웹 서버와 달리, 비즈니스 로직 실행과 애플리케이션 응답 생성을 담당합니다.

## 하는 일

- HTTP 요청 수신
- 애플리케이션 코드 실행
- DB, 외부 API와 연동
- 동적 응답 생성

## 웹 서버와 차이

### 웹 서버
- 정적 파일 전달에 강함
- 예: Nginx, Apache HTTP Server

### WAS
- 애플리케이션 실행에 강함
- 예: Tomcat, Jetty

## 면접 포인트

- 실무에서는 웹 서버와 WAS를 함께 두는 경우가 많으며, 역할 분리가 핵심이라고 설명하면 좋습니다.
