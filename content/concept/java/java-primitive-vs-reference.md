## 기본형(Primitive Type)

Java의 기본형은 값을 직접 다룹니다.

예:
- `int`
- `long`
- `double`
- `boolean`
- `char`

## 참조형(Reference Type)

객체를 가리키는 참조를 다룹니다.

예:
- `String`
- 배열
- 클래스 인스턴스

## 차이의 핵심

- 기본형은 값 자체
- 참조형은 객체에 대한 참조

## 면접 포인트

- 메모리 세부 구현을 과하게 단정하기보다, 값과 참조의 차이를 중심으로 설명하면 좋습니다.
