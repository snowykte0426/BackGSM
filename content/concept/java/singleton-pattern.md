## 싱글턴 패턴

애플리케이션 전체에서 인스턴스를 하나만 가지도록 보장하는 패턴입니다.

## 장점

- 공유 자원 관리
- 생성 비용 절약 가능

## 주의점

- 전역 상태처럼 변질되기 쉬움
- 테스트 어려움
- 결합도 증가 가능

## 면접 포인트

- 무조건 좋은 패턴처럼 말하지 말고, 남용 시 전역 상태 문제가 생길 수 있다고 설명하면 좋습니다.
