## JUnit이란?

Java에서 가장 널리 쓰이는 테스트 프레임워크입니다.

## 자주 쓰는 요소

- `@Test`
- `@BeforeEach`
- `@AfterEach`
- assertion
- parameterized test

## 장점

- 표준에 가까운 생태계
- Spring 테스트와 연동이 좋음
- 러닝커브가 낮음

## 면접 포인트

- JUnit은 기본 테스트 프레임워크, Mockito/MockK는 협력 객체 제어 도구라고 구분하면 좋습니다.
