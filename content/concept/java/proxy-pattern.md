## 프록시 패턴이란?

실제 객체에 대한 접근을 대신하는 대리 객체를 두는 패턴입니다.

## 언제 쓰나?

- 접근 제어
- 지연 로딩
- 로깅
- 캐싱
- 원격 호출 래핑

## 예시

- Hibernate 지연 로딩 프록시
- Spring AOP 프록시

## 면접 포인트

- 프록시는 "대신 접근"이라는 키워드로 설명하면 좋습니다.
