## 한 줄 정의

| | Skill | MCP |
|--|-------|-----|
| **정체** | Claude에게 주는 **프롬프트 템플릿** | Claude가 호출하는 **외부 도구·데이터 서버** |
| **실행 주체** | Claude (LLM이 텍스트 생성) | 외부 프로세스 (코드가 실제 실행) |
| **부수 효과** | 없음 (읽기·생성 전용) | 있음 (파일 쓰기, API 호출, DB 변경) |

---

## Skill이란?

`/` 로 시작하는 **재사용 가능한 프롬프트 슬래시 커맨드**입니다.
마크다운 파일 하나 = 슬래시 커맨드 하나.

```
.claude/skills/
├── commit.md      →  /commit
├── review.md      →  /review
└── deploy.md      →  /deploy
```

### Skill이 잘 맞는 경우

- 반복적으로 쓰는 **지시문·페르소나·출력 형식**을 저장할 때
- Claude의 추론·생성 방식을 바꾸고 싶을 때
- 외부 시스템 연동 없이 텍스트 작업만 할 때

```markdown
<!-- .claude/skills/review.md -->
코드 리뷰어 역할을 맡아줘.
보안 취약점, 성능 병목, 가독성 순서로 검토하고
각 항목을 Severity: High/Mid/Low로 표기해줘.
```

---

## MCP란?

Claude가 **외부 세계와 실제로 상호작용**하게 해주는 서버 프로토콜입니다.
Claude가 MCP 도구를 "호출"하면 별도 프로세스가 실행되고 결과를 돌려줍니다.

```
Claude ──[Tool Call]──► MCP Server ──► 파일 시스템
                                   ──► GitHub API
                                   ──► PostgreSQL
                                   ──► Slack
```

### MCP가 잘 맞는 경우

- 파일·DB·외부 API와 실제로 **읽기/쓰기**가 필요할 때
- 여러 AI 앱에서 **같은 도구를 재사용**하고 싶을 때
- 실시간 데이터(검색 결과, DB 레코드 등)를 가져와야 할 때

---

## 핵심 차이 비교

| 기준 | Skill | MCP |
|------|-------|-----|
| **구현 복잡도** | 마크다운 파일만 작성 | 서버 코드 구현 필요 |
| **실행 환경** | Claude 내부 (LLM) | 별도 프로세스 |
| **외부 연동** | 불가 | 가능 (파일·API·DB) |
| **재사용 범위** | 해당 Claude 인스턴스 | 모든 MCP 지원 클라이언트 |
| **보안 위험** | 낮음 | 높음 (권한 관리 필요) |
| **상태 변경** | 없음 | 있음 |
| **속도** | 빠름 (추가 I/O 없음) | 네트워크/프로세스 오버헤드 |

---

## 선택 기준 플로차트

```
"외부 시스템(파일·DB·API)과 실제로 읽기/쓰기가 필요한가?"
        │
        ├── YES → MCP
        │
        └── NO → "반복적으로 쓰는 지시문·형식인가?"
                        │
                        ├── YES → Skill
                        │
                        └── NO → 그냥 프롬프트로 입력
```

---

## 함께 쓰는 패턴

Skill과 MCP는 배타적이지 않습니다. 함께 조합하면 강력합니다.

```markdown
<!-- .claude/skills/db-review.md -->
너는 DB 성능 전문가야.
MCP로 연결된 PostgreSQL에서 슬로우 쿼리를 조회하고
실행 계획(EXPLAIN ANALYZE)을 분석해서
인덱스 추가 권고안을 테이블로 정리해줘.
```

→ **Skill이 페르소나·출력 형식을 정의**, **MCP가 실제 DB 조회를 수행**

## Hook과의 관계

| | Skill | MCP | Hook |
|--|-------|-----|------|
| **역할** | 프롬프트 재사용 | 외부 도구 연동 | 이벤트 기반 자동 실행 |
| **트리거** | 사용자가 `/` 호출 | Claude가 Tool Use | 특정 이벤트 발생 |
| **코드 실행** | X | O (MCP 서버) | O (쉘 명령) |
