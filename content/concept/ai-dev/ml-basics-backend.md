## 머신러닝 (Machine Learning)

데이터를 보고 **스스로 규칙을 학습**하는 방식. 규칙을 사람이 직접 코딩하지 않습니다.

```
기존 방식: 개발자가 if/else로 규칙 작성
ML 방식:   데이터를 주면 모델이 규칙을 스스로 찾아냄
```

**백엔드에서 마주치는 ML 적용 사례**

| 기능 | 학습 방식 |
|------|---------|
| 스팸 메일 필터 | 지도 학습 |
| 상품 추천 | 비지도 학습 / 지도 학습 |
| 이상 거래 탐지 | 비지도 학습 |
| 게임 AI, 자동 배포 최적화 | 강화 학습 |
| 이미지 인식, LLM | 딥러닝 |

---

## 지도 학습 (Supervised Learning)

**정답이 있는 데이터**로 학습합니다. 입력 → 정답 쌍을 대량으로 제공.

```
입력(X)           정답(Y)
─────────────────────────
이메일 텍스트  →  스팸 / 정상
거래 금액, 위치 →  사기 / 정상
사용자 행동 로그 → 이탈 / 유지
```

### 백엔드 활용 예

- **분류(Classification)**: 스팸 탐지, 사기 거래 탐지, 콘텐츠 검열
- **회귀(Regression)**: 상품 가격 예측, 트래픽 예측 (오토스케일링 근거)

### API 서빙 구조

```
Client → REST API → ML 모델 (추론) → 예측 결과 반환
                  ↑
         사전 학습된 모델 로드 (파일 or 모델 서버)
```

---

## 비지도 학습 (Unsupervised Learning)

**정답 없이** 데이터 자체의 패턴을 찾습니다.

```
입력(X)만 존재, 정답(Y) 없음
→ 모델이 스스로 구조·그룹을 발견
```

### 주요 기법

| 기법 | 설명 | 백엔드 활용 |
|------|------|----------|
| **클러스터링** | 비슷한 것끼리 묶기 | 사용자 세그먼트, 로그 이상 그룹화 |
| **차원 축소** | 고차원 데이터를 압축 | 임베딩, 벡터 DB 최적화 |
| **이상 탐지** | 정상 패턴 학습 → 이탈 탐지 | 서버 이상 감지, DDoS 탐지 |

### 백엔드 활용 예

- **이상 탐지**: 정상 트래픽 패턴 학습 → 비정상 요청 자동 차단
- **사용자 클러스터링**: 행동 패턴으로 그룹화 → 그룹별 다른 비즈니스 로직 적용

---

## 강화 학습 (Reinforcement Learning)

**보상/패널티 신호**로 시행착오를 통해 학습합니다. 정답 데이터 없이 환경과 상호작용.

```
Agent(에이전트) → 행동 → 환경 → 보상/패널티
       ↑___________________________|
```

### 백엔드 관련 활용

- **광고 입찰 최적화**: 클릭률 보상 신호로 입찰 전략 자동 학습
- **추천 시스템**: 사용자 반응(클릭·구매)을 보상으로 실시간 개선
- **오토스케일링**: 서버 비용·응답속도를 보상 함수로 설정해 스케일 정책 최적화
- **LLM RLHF**: ChatGPT 등이 사람 피드백으로 품질을 높이는 데 사용

---

## 딥러닝 (Deep Learning)

**인공 신경망(Neural Network)** 기반의 ML. 층(Layer)을 깊게 쌓아 복잡한 패턴을 학습합니다.

```
입력층 → [히든 레이어 × N] → 출력층
         (깊을수록 = "딥")
```

### 왜 딥러닝이 주류가 됐나?

- 이미지·텍스트·음성 같은 **비정형 데이터** 처리에 탁월
- 대규모 데이터 + GPU 연산으로 기존 ML을 압도
- LLM(GPT, Claude), 이미지 생성(Stable Diffusion)의 기반

### 백엔드 개발자가 알아야 할 딥러닝 서빙

```
학습(Training)    →  GPU 클러스터 (AI 팀 담당)
추론(Inference)   →  백엔드 API로 서빙 (백엔드 팀 담당)
                      ↓
               모델 서버 (TorchServe, Triton, BentoML)
               또는 외부 API (OpenAI, Anthropic)
```
