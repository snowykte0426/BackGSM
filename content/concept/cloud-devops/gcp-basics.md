## GCP란?

Google Cloud Platform의 약자로, Google이 제공하는 클라우드 서비스 플랫폼입니다.

## 자주 나오는 대표 서비스

- Compute Engine: VM
- GKE: Kubernetes
- Cloud Storage: 객체 스토리지
- Cloud SQL: 관리형 RDB
- BigQuery: 대용량 분석

## 특징

- 데이터/분석 서비스 강점
- Kubernetes 친화 이미지가 강함

## 면접 포인트

- GCP는 단순히 "또 다른 클라우드"가 아니라, GKE와 BigQuery처럼 강한 대표 서비스와 함께 설명하면 좋습니다.
