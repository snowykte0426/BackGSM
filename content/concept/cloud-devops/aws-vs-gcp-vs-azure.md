## 비교 기준

- 컴퓨트
- 스토리지
- 관리형 DB
- Kubernetes
- IAM
- 모니터링/운영 도구
- 조직의 기존 생태계

## 흔한 이미지

- AWS: 가장 넓은 생태계와 서비스 범위
- GCP: 데이터/분석, Kubernetes 강점
- Azure: Microsoft 친화성과 기업 환경 강점

## 면접 포인트

- "어느 클라우드가 최고다"보다, 조직의 기술 스택과 운영 역량에 따라 선택 기준이 달라진다고 설명하면 좋습니다.
