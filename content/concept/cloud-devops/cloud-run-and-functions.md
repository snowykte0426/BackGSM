## 서버리스란?

서버를 직접 관리하지 않고, 실행 단위 중심으로 애플리케이션을 배포하는 방식입니다.

## 예시

- GCP Cloud Run
- Cloud Functions
- Azure Functions
- AWS Lambda

## 장점

- 운영 부담 감소
- 자동 확장
- 작은 서비스에 적합

## 단점

- 콜드 스타트
- 제약된 실행 환경
- 복잡한 장기 연결 서비스엔 불리할 수 있음

## 면접 포인트

- 서버리스는 "서버가 없다"가 아니라, 서버 운영을 추상화한 모델이라고 설명하면 좋습니다.
