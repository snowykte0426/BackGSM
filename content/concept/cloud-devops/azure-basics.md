## Azure란?

Microsoft가 제공하는 클라우드 플랫폼입니다.

## 자주 나오는 대표 서비스

- Virtual Machines
- AKS
- Azure Blob Storage
- Azure SQL Database
- Azure Functions

## 특징

- Microsoft 생태계와 강한 연결
- 기업 환경에서 자주 사용

## 면접 포인트

- Azure는 Microsoft 제품군과의 통합성과 엔터프라이즈 친화성을 함께 말하면 좋습니다.
