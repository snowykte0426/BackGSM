## 객체 스토리지

파일을 객체 단위로 저장합니다.
예: S3

## 블록 스토리지

디스크 블록 단위로 저장하며, VM 디스크나 DB 저장소에 자주 사용됩니다.

## 파일 스토리지

공유 파일 시스템처럼 디렉터리/파일 구조로 접근합니다.

## 면접 포인트

- 세 가지를 하나의 "스토리지"로 뭉개지 않고, 워크로드별로 구분해서 설명하면 좋습니다.
