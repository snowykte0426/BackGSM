## GitHub Actions란?

GitHub 저장소 이벤트를 기반으로 자동화 워크플로를 실행하는 CI/CD 도구입니다.

## 장점

- GitHub와 긴밀한 통합
- YAML 기반 선언형 파이프라인
- 빠른 설정

## 언제 좋은가?

- GitHub 중심 개발 흐름
- 비교적 간단한 파이프라인

## 면접 포인트

- Jenkins와 비교해 운영 단순성과 통합성이 강점이라고 설명할 수 있습니다.
