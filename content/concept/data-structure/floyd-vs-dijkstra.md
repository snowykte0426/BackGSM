## 다익스트라

- 한 시작점에서 다른 정점들까지의 최단 경로
- 음수 가중치가 없을 때 사용
- 우선순위 큐 사용 시 보통 `O((V + E) log V)`

## 플로이드-워셜

- 모든 정점 쌍 최단 경로
- 동적 프로그래밍 기반
- 시간 복잡도 `O(V^3)`

## 어떤 상황에서 고르나?

### 다익스트라
- 특정 출발점 기준 최단 거리
- 희소 그래프
- 빠른 단일 출발점 계산이 중요할 때

### 플로이드-워셜
- 모든 정점 쌍 결과가 필요할 때
- 정점 수가 아주 크지 않을 때
- 구현이 비교적 단순함

## 점근 표기법

- Big-O: 상한
- Big-Omega: 하한
- Big-Theta: 상한과 하한이 같은 tight bound

## 면접 포인트

- "둘 다 최단 경로"가 아니라 문제 범위가 다릅니다.
- 하나는 single-source, 하나는 all-pairs라는 점을 먼저 말하면 좋습니다.
