## RFC란?

Request for Comments의 약자로, 인터넷 기술 표준과 관련 문서를 정리한 문서 시리즈입니다.

## 왜 중요한가?

- 프로토콜 정의의 근거 문서
- 표준 동작을 확인하는 기준
- 구현 간 호환성의 기반

## 예시

- HTTP 관련 RFC
- TLS 관련 RFC
- URI, DNS, SMTP 관련 RFC

## 면접 포인트

- RFC는 단순 블로그 글이 아니라, 인터넷 표준과 동작 정의의 공식적인 참고 문서라고 설명하면 좋습니다.
