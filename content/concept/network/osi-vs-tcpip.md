## OSI 7계층

네트워크 통신을 7개 계층으로 나누어 설명하는 참조 모델입니다.

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## TCP/IP 모델

실제 인터넷 프로토콜 구조를 설명할 때 더 자주 쓰이는 모델입니다.

보통:
- Link
- Internet
- Transport
- Application

## 왜 둘 다 배우나?

- OSI는 개념적 이해에 좋음
- TCP/IP는 실제 프로토콜 스택 설명에 더 가깝습니다

## 면접 포인트

- OSI는 참조 모델, TCP/IP는 실제 인터넷 구조 설명에 더 가깝다고 말하면 좋습니다.
