## 프로세스 메모리 구조

일반적으로 프로세스 메모리는 다음처럼 나눠 생각합니다.

- Code(Text)
- Data
- Heap
- Stack

## 각각의 의미

- Code: 실행 코드
- Data: 전역 변수, static 데이터
- Heap: 동적 할당 메모리
- Stack: 함수 호출 정보, 지역 변수

## 면접 포인트

- Stack과 Heap 차이를 설명할 수 있으면 자주 이어지는 질문에 대응하기 좋습니다.
