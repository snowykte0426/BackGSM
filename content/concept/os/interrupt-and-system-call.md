## 인터럽트

CPU가 현재 작업 흐름을 잠시 멈추고 다른 중요한 이벤트를 처리하게 만드는 신호입니다.

예:
- 입출력 완료
- 타이머
- 하드웨어 이벤트

## 시스템 콜

사용자 프로그램이 운영체제 커널 기능을 요청하는 인터페이스입니다.

예:
- 파일 열기
- 네트워크 소켓 생성
- 프로세스 생성

## 면접 포인트

- 인터럽트는 CPU 흐름을 바꾸는 사건, 시스템 콜은 사용자 프로그램이 커널 기능을 요청하는 통로라고 설명하면 좋습니다.
