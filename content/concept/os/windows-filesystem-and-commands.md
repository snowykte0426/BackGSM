## Windows 파일 시스템

대표적으로 NTFS를 사용합니다.

특징:
- 권한 관리
- 저널링
- 압축/암호화 지원

## 자주 보는 Windows 명령어

- `dir`: 파일 목록
- `cd`: 디렉터리 이동
- `copy`: 파일 복사
- `move`: 파일 이동
- `del`: 파일 삭제
- `ipconfig`: 네트워크 정보 확인
- `tasklist`: 프로세스 목록

## 면접 포인트

- 리눅스처럼 폭넓게 묻기보다, 기본 명령과 파일 시스템 이해 정도를 보는 경우가 많습니다.
