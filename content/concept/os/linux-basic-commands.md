## 자주 쓰는 명령어

- `ls`: 목록 보기
- `cd`: 이동
- `pwd`: 현재 경로
- `cp`: 복사
- `mv`: 이동/이름 변경
- `rm`: 삭제
- `cat`: 파일 보기
- `grep`: 문자열 검색
- `ps`: 프로세스 확인
- `top` / `htop`: 시스템 상태 확인
- `chmod`: 권한 변경
- `chown`: 소유자 변경

## 면접 포인트

- 명령어 이름만 외우기보다, 파일/프로세스/권한 관점으로 분류해 말하면 더 좋습니다.
