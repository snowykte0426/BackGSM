**프로세스(Process)**는 실행 중인 프로그램의 인스턴스로, 독립된 메모리 공간(코드, 데이터, 힙, 스택)을 가집니다.

**스레드(Thread)**는 프로세스 내의 실행 단위로:
- **공유**: 코드, 데이터, 힙, 파일 디스크립터
- **독립**: 스택, 레지스터, PC(Program Counter)

### 컨텍스트 스위칭 비용
- 프로세스 간 전환: 메모리 공간 전환 필요 → 비용 大
- 스레드 간 전환: 같은 메모리 공간 → 비용 小

### 안전성
멀티프로세스는 독립 메모리로 한 프로세스 crash가 다른 프로세스에 영향 없음.
멀티스레드는 자원 공유로 인해 **경쟁 조건(Race Condition)** 주의 필요.
