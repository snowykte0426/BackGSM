**OSWorld**는 AI 모델이 실제 데스크톱 환경에서 GUI 앱을 조작하는 능력을 측정하는 벤치마크입니다.

GPT-5.4의 **네이티브 Computer Use(컴퓨터 직접 사용)** 기능:
- 브라우저 탐색, 파일 관리, 스프레드시트 편집 등 실제 GUI 작업 수행
- 여러 소프트웨어에 걸친 멀티 스텝 워크플로 자동화
- 2026년 3월 출시 시 OSWorld 75.0% 달성

이 능력은 단순 코드 생성을 넘어 **실제 사람처럼 컴퓨터를 다루는 에이전트**로의 진화를 의미합니다.
