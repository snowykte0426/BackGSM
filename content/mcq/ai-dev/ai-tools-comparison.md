- **GitHub Copilot**: VS Code, JetBrains 등 IDE에 깊이 통합된 **인라인 자동완성·채팅** 도구. GPT-4o, Claude, Gemini 등 다중 모델 지원.
- **Claude Code**: Anthropic의 **CLI 기반 에이전트** 도구. 파일 읽기/쓰기, 터미널 명령 실행, git 조작, MCP 서버 연동까지 자율 수행. VS Code·JetBrains 확장도 제공.

Claude Code는 "무엇을 해야 하는지"를 목표로 주면 스스로 계획하고 실행하는 **에이전트적 특성**이 핵심 차이입니다.
